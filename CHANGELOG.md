# 更新日志

## v1.3.0 (Build 5) — 2026-07-10

### 🆕 新增功能

- **💬 遮罩层微信按钮**：一键打开微信处理紧急聊天消息
- **🔋 定时范围外省电模式**：完全停止光线传感器、WakeLock、屏幕监听、无障碍窗口检测，仅保留定时轮询
- **📊 通知栏动态状态**：「电话使用中」「微信使用中」「定时暂停中」「光线太暗，已锁定」「当前亮度：xxx lux」

### 🔧 改进

- **微信前台检测**：通过无障碍 `getWindows()` 兜底检测，解决 Flyme 等 ROM 不触发微信窗口事件的问题
- **定时精度优化**：轮询改为动态间隔，精确计算到下一个边界时间，最大延迟从 60s 降至 10s
- **电话/微信不受锁定状态限制**：高亮度下使用电话/微信时亮度突降不会误弹遮罩
- **状态同步修复**：电话/微信活跃时取消待处理的锁定定时器，防止状态机不同步
- **遮罩恢复同步**：无障碍防抖恢复遮罩时同步 LightMonitorService 锁定状态，光线恢复后自动解锁

### ✅ 其他

- 升级版本号至 1.3.0
- Debug 日志默认关闭（Release 构建）
